\# Sistema de Casa de Leilões



\##  Descrição do Projeto



Sistema informatizado desenvolvido para gerenciar as operações de uma casa de leilões, incluindo cadastro de itens, controle de lances, gestão de clientes e relatórios de vendas.



\##  Tecnologias Utilizadas



\- \*\*Java\*\* - Linguagem de programação principal

\- \*\*MySQL\*\* - Sistema de gerenciamento de banco de dados

\- \*\*JDBC\*\* - Conectividade com banco de dados

\- \*\*Swing\*\* - Interface gráfica do usuário



\##  Funcionalidades



\- Cadastro de produtos para leilão

\- Registro de clientes/licitantes

\- Sistema de lances

\- Controle de vendas

\- Geração de relatórios

\- Autenticação de usuários



\##  Como Executar



1\. Clone o repositório

2\. Importe o banco de dados MySQL (arquivo SQL incluído)

3\. Configure a conexão com o banco de dados

4\. Execute a aplicação através da classe principal
